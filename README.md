# Movie-Review-Sentiment-Analysis
Sentiment Analysis of Movie Reviews

A machine learning project that predicts whether a movie review is positive or negative using:
- TF-IDF Vectorization
- Logistic Regression
- Flask web interface

## Features
✔ ML model trained on IMDB dataset  
✔ Flask web app to enter reviews  
✔ Sentiment shown as Positive/Negative  

## Tech Used
- Python
- Flask
- Scikit-learn
- Pandas
- Jinja2 Templates
